# dialy [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> A PPP dialer

## Installation

```sh
$ npm install --save dialy
```

## Usage

```js
var dialy = require('dialy');

dialy('Rainbow');
```
